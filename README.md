## 论文自动化工具
**开发者：工程研究院绿色低碳技术研究所**  
  
本工具用于监测[Arxiv](https://arxiv.org/)上最新论文的动态，并根据用户设定的主题、关键词等进行自动化检索、下载、筛选、总结分析并形成简报，其中筛选和总结分析采用了Moonshot AI的大模型[Kimi Chat](https://kimi.moonshot.cn/)。
